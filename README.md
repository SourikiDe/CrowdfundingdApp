# 🚀 Crowdfunding dApp on Stellar (Soroban)

## 📌 Project Description
This project is a decentralized crowdfunding application built using Soroban smart contracts on the Stellar blockchain. It enables users to create fundraising campaigns and allows others to contribute funds securely and transparently.

## ⚙️ What it does
- Users can create crowdfunding campaigns with a funding goal.
- Other users can contribute funds to campaigns.
- Contributions are recorded on-chain for transparency.
- Anyone can query campaign details and contributions.

## ✨ Features
- 🪙 Create Campaigns with funding goals
- 💸 Contribute to campaigns
- 📊 Track contributions on-chain
- 🔐 Decentralized & trustless system
- ⚡ Built on Soroban (fast & efficient smart contracts)
